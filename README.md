# 📈 Stock Analyser using LSTMs

> A deep learning project that predicts and analyzes stock price trends using **Long Short-Term Memory (LSTM)** networks.  
> Built to help traders, investors, and analysts make data-driven decisions based on historical market patterns.

---

## 🧠 Overview

**Stock Analyser** is an AI-driven forecasting system designed to analyze stock market behavior and predict future prices using **Recurrent Neural Networks (RNNs)** — specifically **LSTMs**, which are highly effective in modeling time series data.  

This project demonstrates:
- How to preprocess financial time series data.
- How to design and train an LSTM model.
- How to visualize and evaluate its predictive performance.

---

## 🚀 Key Features

- 📊 **Stock Data Fetching** — Automatically fetches historical price data from APIs like Yahoo Finance.  
- 🔍 **Data Preprocessing** — Scaling, feature extraction, and window-based input generation.  
- 🤖 **Deep Learning Model** — LSTM-based architecture built with TensorFlow/Keras.  
- 📈 **Prediction Visualization** — Compare predicted vs. actual prices interactively.  
- 🧩 **Evaluation Metrics** — MSE, RMSE, and correlation-based performance reports.  
- 💡 **Customizable Inputs** — Change ticker symbols, time intervals, and model parameters easily.

---

## 🧰 Tech Stack

### Core
- **Python 3.8+**
- **TensorFlow / Keras** — Neural network modeling.
- **NumPy, Pandas** — Data manipulation and preprocessing.
- **Matplotlib, Seaborn** — Visualization.
